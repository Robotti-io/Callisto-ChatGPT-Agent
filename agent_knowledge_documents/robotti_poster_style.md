# Robotti Poster Style Guide

This guide defines the required structure, tone, and layout for all HTML posters generated by Callisto – A Robotti Security Awareness Companion.

## Layout Rules

- Two-column grid layout (`.container`): 2/3 main content, 1/3 quick tips
- Class names must match the HTML template (e.g., `.bottom-highlight`, `.quick-tips`)
- All CSS must be embedded inline (no external styles)
- The poster uses an emotionally aligned color palette (see `robotti_color_system.md`)

## Required Elements

- `<h1>` title with emoji (color-themed header bar)
- `<h2>` intro context line *immediately after the title*
- `<p class="quote">` emotional hook (must follow the intro context)
- 2–3 `<h2>` sections with `<p>` + `<ul>` content blocks
- A `.quick-tips` box labeled `"📌 Quick Tips"`, structured with 2–3 titled sections (e.g., “Drafting”, “Reviewing”)  
  Each section should use a `<h4>` title + `<ul>` with 1–2 action tips
- A `.bottom-highlight` block for closing message
- A footer with “Further Reading” links

## Tone and Voice

- Use metaphor or story as the hook
- Never shame, always educate
- Avoid compliance tone — this is about clarity, not command
- Emotional pacing is critical: calm > urgent

## Framing-Specific Styling Cues

Each title framing pattern influences the structure and rhythm of the poster.  
The GPT should adapt content shape — not just emotional tone.

- **Juxtaposition** → Clear contrast, fast pacing, A/B logic  
  Use punchy sentence pairs and highlight what *seems safe vs. what isn’t*.

- **Consequence Frame** → Cause → effect → responsibility  
  Use chain logic and organizational implications (e.g., “You clicked, now…”).

- **Emotional Hook** → Start with a moment or feeling  
  Lead with a *story*, not a checklist. Delay the tips. Let the emotion land.
